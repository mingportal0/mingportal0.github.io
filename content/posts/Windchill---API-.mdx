# 문서 리스트

* Javadoc : Windchill Javadoc
* JSDoc : Windchill JavaScript Documentation
* OData REST APIs : API documentation for Windchill REST Services. These REST APIs are based on the OData specification.
* REST APIs (Legacy) : API documentation for legacy Windchill REST APIs. These REST APIs are based on RESTEasy. It is recommended to use the OData REST APIs available in Windchill REST Services.
