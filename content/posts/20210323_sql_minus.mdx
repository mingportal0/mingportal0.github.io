B에 속하지 않은 A

![](/uploads/image-20210323171456994.png)

```sql
SELECT *                                           
FROM (                                             
    SELECT *                                       
    FROM muserm A MINUS (                          
        SELECT m.* FROM sgroupuserlinkm g, muserm m
        WHERE GROUPCODE = ?                        
        AND g.USERID = m.USERID                    
    )                                              
)                               
```
