# VO와 DTO

Value Object와 Data Transfer Object는 거의 비슷한 용어이다.  하지만 인터넷 상 찾아보면 읽기만 가능한 DTO가 VO라고 말하기도 한다. 일반적으로 로직이 없고 Getter, Setter만 있다.

# DAO

Data Acess Object. DB에 접근하는 객체. 웹서버가 DB에 접근할 때 Connection Pool을 생성하고 다쓰면 다시 반환하는데 이것을 DAO에서 전담하여 DB접근을 효율적으로 하기 위해 사용한다.
