# Preferences 설정

## Heap 메모리 표시

`Window - Preferences - General`

## 주석

`Window - Preferences - Java - Code Style - Code Templates`

### File

```java
/**
  * @FileName : ${file_name}
  * @Project : ${project_name}
  * @Date : ${date}
  * @작성자 : ${user}
  * @프로그램 설명 : 
  */
```

### Method

```java
/**
  * @Method Name : ${enclosing_method}
  * @작성일 : ${date}
  * @작성자 : ${user}
  * @Method 설명 : 
  * ${tags}
  */
```
