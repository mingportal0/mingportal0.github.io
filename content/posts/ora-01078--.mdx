oracle sid가 wind와 wind2로 나누어져 있는 환경이고 서비스를 재시작하는 상황이었다. 서비스는 성공적으로 재시작되었지만 wind2의 데이터베이스가 안올라오는 상황이 발생했다.

그래서 sqlplus에서 wind2로 접속한 후 데이터베이스를 시작해보았는데 다음과 같은 에러가 발생하였다.

```
SQL > startup
ORA - 01078: failure in processing system parameters
ORA - 00821: Specified value of sga_target 16M is too small, needs to be at least 128M
```
