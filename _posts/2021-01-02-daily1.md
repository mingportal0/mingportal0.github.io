### 1. 주택 종류

+ 국민주택 : 주거전용면적이 85m2 이하 (수도권) 이며 국가, 지방자치 단체, LH 및 지방공사가 건설 또는 지원하여  건설 / 리모델링하는 주택
+ 민영주택 : 국민주택을 제외한 주택



### 2. 국민주택 자격

+ 1순위 (청약통장 가입기간, 납입금 )

1. 투기과열지구, 청약과열지역 : 2년, 24회

2. 위축지역 : 1개월, 1회

3. 그 외  수도권 지역일 경우 1년, 12회

+ 2순위 : 1순위에 해당되지 않는 경우



### 3. 특별공급

**1) 신혼부부**

+ 대상 : 85m2 이하 국민주택, 민영주택,  분양주택, 분양전환임대주택인 국민주택(공공주택 특별법 적용) 투기과열지구 내 분양가 9억 초과 주택은 제외.

+ 공급 물량

  1. 국민주택, 민영주택 : 건설량의 20% 내

  2. 공공주택 적용 국민주택 : 건설량의 30% 내

+ 대상자

  0. 공통 : 공고일 기준 혼인기간이 7년 이내

  1. 공공주택 적용 국민주택 : 예비신혼부부

+ 자격

  1. 무주택세대구성원

  2. 소득기준

     1. 국민주택, 민영주택

        + 기준소득기준에 해당하는 신청자에게 특별공급 배정물량의 75%를 우선 공급. 경쟁이 있을 시 우선순위에 따라 입주자를 선정

        + 기준소득기준 (75%)

          1. 배우자 소득이 없는 신혼부부 : 월평균소득 100%. 3인 이하 5,554,983원

          2. 배우자 소득이 있는 신혼부부 : 월평균소득 120%, 3인 이하 6,665,980원 (부부 중 1인 소득이 월평균소득 100%를 넘으면 안됨.)

        + 기준소득기준(25%)

          1. 배우자 소득이 없는 신혼부부
             + 기본 : 월평균소득 100% ~ 120% (5,554,984 ~ 6,665,980원)
             + 6억~9억 이하 주택 생애최초 : 월평균소득 100%~130 이하 (5,554,984 ~ 7,221,478원)

          2. 배우자 소득이 있는 신혼부부
             + 기본 : 월평균소득 100% ~ 130% (5,554,984 ~ 7,221,478원)
             + 6억~9억 이하 주택 생애최초 : 월평균소득 100%~140 이하 (5,554,984 ~ 7,776,976원)

     2. 공공주택 적용 국민주택
        + 월평균소득 100% 이하(5,554,984), 배우자 소득이 있는 경우 120% 이하 (6,665,980원)

  3. 자산기준

     2. 공공주택 적용 국민주택 : 부동산 215,500,000원 이하, 자동차 27,640,000원 이하

  4. 청약통장

     0 . 공통 : 가입기간 6개월, 납입금 6회

     1. 국민주택, 민영주택 : 지역별 예치금액 납부

  5. 당첨자 발표일이 동일한 주택 전체에 대해 1세대 1인만 청약신청 가능.

+ 선정방법

  + 우선순위

    + 1순위 : 현재 신청자와 배우자 사이 출산한 미성년인 자녀가 있는 경우.

    + 2순위 : 1순위에 해당하지 않는 경우

    + 같은 순위 내 우선순위

      1. 국민주택, 민영주택 : 해당 주택건설지역 거주자, 자녀수가 많은 자(만약 같을 시  추첨으로 결정함.)

      2. 공공주택 적용 국민주택 : 가점이 높은 자 순으로 선정

+ 증명서류

  + 가족관계증명서 : 배우자 및 자녀에 관한 사실 확인
  + 혼인관계증명서 : 혼인신고일 확인
  + 임신진단서
  + 청약통장가입확인서
  + 주민등록등본 : 세대주 및 세대구성원 사실 확인
  + 주민등록초본 : 거주지 및 거주기간 등 확인
  + 건강(의료)보험증 사본 : 세대주 및 세대구성원의 직업 유무
  + 소득입증서류 : 전년도 근로소득원천징수영수증, 재직증명서

**2) 생애최초**

+ 대상 : 국민주택, 민영주택(85m2 이하)

+ 공급물량 : 국민주택 건설량의 25% 내. 민영주택 건설량의 7%(공공택지 15%) 내

+ 대상자 (다음의 요건을 모두 충족하는 경우)

  0. 공통
     + 일반공급 1순위
     + 공고일 기준 혼인중이거나 자녀가 있는 경우
     + 공고일 기준 근로자 또는 자영업자로서 5년 이상 소득세를 납부한 경우
     + 투기과열지구 또는 청약과열지구의 주택에 특공 청약 시 과거 5년 이내 주택에 당첨된 자가 세대에 속한 자는 대상이 아님

  1. 국민주택
     + 청약통장 예치금 600만원 이상

+ 자격

  1. 무주택세대구성원

  2. 소득기준

     1. 국민주택 : 월평균소득 100% 이하

     2. 민영주택 : 월평균소득 130% 이하

  3. 청약통장

     0. 공통 : 가입기간 24개월

     1. 국민주택 : 1순위자. 납입금 24회. 예치금 600만원.

     2. 민영주택 : 1순위자. 지역별 예치금 기준 충족.

  4. 자산기준(국민주택만 해당. 해당 안되는 국민주택도 있음.) : 부동산 215,500,000원 이하, 자동차 27,640,000원 이하

  5. 당첨자 발표일이 동일한 주택 전체에 대해 1세대 1인만 청약신청 가능.

+ 선정방법

  + 경쟁이 있을 경우 추첨으로 선정

+ 증명서류

  + 가족관계증명서 : 배우자 및 자녀에 관한 사실 확인
  + 청약통장가입확인서
  + 민등록등본 : 세대주 및 세대구성원 사실 확인
  + 주민등록초본 : 거주지 및 거주기간 등 확인
  + 건강(의료)보험증 사본 : 세대주 및 세대구성원의 직업 유무
  + 소득입증서류 : 전년도 근로소득원천징수영수증, 재직증명서
