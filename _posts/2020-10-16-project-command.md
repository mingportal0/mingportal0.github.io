### VSCODE 단축키

+ ctrl + F2 : 같은 단어 한번에 수정
+ 단어에 F2 : 단어 정의 수정 (위와 비슷)
+ alt + up/down : 현재 행 위/아래로 이동
+ ctrl + B : 좌측 사이드바 show/hide
+ alt + shift + up/down : 현재 행 위/아래로 복사
+ ctrl + enter : 커서 아래로 이동
